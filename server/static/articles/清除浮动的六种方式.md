# 清除浮动的六种方式

### 一、为兄弟元素设置clear样式

+ 基本语法

  ~~~css
  clear:both|left|right|none|inherit;
  ~~~

+ 缺点：会导致最后一个子盒子的margin失效

+ 代码解析

  ~~~
  设置元素某侧不允许存在浮动元素
  ~~~

+ 代码操作

  ~~~
  最后一个子盒子添加clear:both属性，并且不浮动，即float:none
  ~~~


+ 代码实例

  ~~~html
  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <title>Title</title>
      <style>
          .box {
              width: 300px;
              background-color: black;
              
          }
          .item1 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: red;
          }
          .item2 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: pink;
          }
          .item3 {
              width: 200px;
              height: 200px;
              background-color: aquamarine;
              clear: both;
          }
      </style>
  </head>
  <body>
      <div class="box">
          <div class="item1"></div>
          <div class="item2"></div>
          <div class="item3"></div>
      </div>
  </body>
  </html>
  ~~~

### 二、空标签清除浮动

+ 代码操作

  ~~~
  在浮动元素的最后，增加一个新标签，这个新标签是浮动元素的兄弟级元素，之后为这个标签设置clear属性
  ~~~


+ 缺点

  ~~~
  会添加大量无语义空标签，结构与表现没有分离，不利于维护
  ~~~


+ 代码实例

  ~~~html
  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <title>Title</title>
      <style>
          .box {
              width: 300px;
              background-color: black;
              border: 1px solid black;
              
          }
          .item1 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: red;
          }
          .item2 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: pink;
          }
          .item3 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: aquamarine;
          }
      </style>
  </head>
  <body>
      <div class="box">
          <div class="item1"></div>
          <div class="item2"></div>
          <div class="item3"></div>
          <div style="clear: both;"></div>
      </div>
  </body>
  </html>
  ~~~

### 三、br标签清除浮动

+ 操作

  ~~~
  br标签清除浮动的方法类似空标签清除浮动，在浮动元素后面添加一个br标签，在br标签中设置属性clear，并赋值all,即：<br clear="all">
  ~~~


+ 缺点

  ~~~
  结构与表现未分离
  ~~~


+ 代码实例

  ~~~html
  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <title>Title</title>
      <style>
          .box {
              width: 300px;
              background-color: black;
              border: 1px solid black;
              
          }
          .item1 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: red;
          }
          .item2 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: pink;
          }
          .item3 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: aquamarine;
          }
      </style>
  </head>
  <body>
      <div class="box">
          <div class="item1"></div>
          <div class="item2"></div>
          <div class="item3"></div>
          <br clear="all">
      </div>
  </body>
  </html>
  ~~~

### 五、父元素设置overflow:hidden或auto

  ~~~

### 六、利用after伪元素清除浮动

+ 操作

  ~~~
  为当前父元素添加伪元素，设置功能代码
  ~~~


+ 缺点

  ~~~
  如果使用不合理，有可能造成代码量增加，还要注意浏览器的支持情况
  ~~~


+ 代码实例

  ~~~html
  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <title>Title</title>
      <style>
          .box {
              width: 300px;
              background-color: black;
              border: 1px solid black;
          }
          .box:after {
              content: '\200B';
              clear: both;
              display: block;
              height: 0;
          }
          .box {
              *zoom: 1
          }
          .item1 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: red;
          }
          .item2 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: pink;
          }
          .item3 {
              float: left;
              width: 200px;
              height: 200px;
              background-color: aquamarine;
          }
  
      </style>
  </head>
  <body>
      <div class="box">
          <div class="item1"></div>
          <div class="item2"></div>
          <div class="item3"></div>
      </div>
  </body>
  </html>
  ~~~
